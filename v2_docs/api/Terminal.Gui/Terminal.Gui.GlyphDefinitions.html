    Class GlyphDefinitions

    Defines the standard set of glyphs used to draw checkboxes, lines, borders, etc...
    Inheritance
    object
    GlyphDefinitions
    Remarks

    Access with Glyphs (which is a global using alias for Glyphs).

    The default glyphs can be changed via the ConfigurationManager. Within a config.json file The Json property name is the property name prefixed with "Glyphs.".

    The JSon property can be either a decimal number or a string. The string may be one of: - A unicode char (e.g. "☑") - A hex value in U+ format (e.g. "U+2611") - A hex value in UTF-16 format (e.g. "\\u2611")
