    Declaration
    public TreeTableSource(TableView table, string firstColumnName, TreeView<T> tree, Dictionary<string, Func<T, object>> subsequentColumns)
    Parameters
    Type Name Description
    TableView table The table this source will provide data for.
    string firstColumnName Column name to use for the first column of the table (where the tree branches/leaves will be rendered.
    TreeView<T> tree The tree data to render. This should be a new view and not used elsewhere (e.g. via Add(View)).
    Dictionary<string, Func<T, object>> subsequentColumns Getter methods for each additional property you want to present in the table. For example: 
     new () {
    { "Colname1", (t)=>t.SomeField},
    { "Colname2", (t)=>t.SomeOtherField}
}
